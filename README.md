# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing sales data to extract meaningful insights and build an interactive dashboard using Power BI. The goal is to identify sales trends, profit patterns, and regional performance for better business decision-making.

---

## 🛠️ Tools & Technologies 

* Python (Pandas, NumPy, Matplotlib)
* Power BI

---

## 📂 Dataset

* Source: Sample Superstore Dataset
* Contains information about orders, sales, profit, discount, and regions

---

## ⚙️ Process

1. Data Cleaning using Python (handled missing values, removed duplicates)
2. Data Transformation using Pandas
3. Exported clean dataset for visualization
4. Dashboard creation in Power BI

---

## 📊 Key Insights

* Identified top-performing categories and regions
* Analyzed monthly sales trends
* Detected loss-making products
* Evaluated profit distribution
